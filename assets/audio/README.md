# AnySkill — Audio Branding Assets

Place the following 4 MP3 files in this directory before building:

| File | Description | Psychology |
|------|-------------|-----------|
| `wealth_crystal.mp3` | High-freq crystal "clink" (150–250ms) | Reward / Luxury |
| `solution_snap.mp3` | Vacuum-seal / lock-in click (100–180ms) | Tension Relief |
| `opportunity_pulse.mp3` | Soft organic double-tap (200–300ms) | Curiosity / Urgency |
| `growth_ascend.mp3` | 4-note ascending scale (400–600ms) | Achievement / Dopamine |

## Recommended specs
- Format: MP3 (44.1 kHz, 128 kbps)
- Duration: keep under 600ms for maximum psychological impact
- Normalise to -3 dBFS peak
