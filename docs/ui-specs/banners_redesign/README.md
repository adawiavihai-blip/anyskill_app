# עיצוב מחודש - טאב ניהול באנרים

חבילה מלאה לשיפור טאב "באנרים" במערכת. כולל מוקאפים אינטראקטיביים, מפרט מוצר מפורט, ופרומפט מוכן ל-Claude Code.

## מה יש בחבילה

```
banners_redesign/
├── README.md                                    ← הקובץ הזה
├── mockups/
│   ├── 01_banners_list.html                     ← המסך הראשי (טבלת באנרים)
│   └── 02_provider_carousel_builder.html        ← מסך בניית קרוסלת נותני שירות
└── docs/
    ├── 01_product_spec.md                       ← מפרט מוצר מלא
    └── 02_claude_code_prompt.md                 ← פרומפט מפורט ל-Claude Code
```

## איך להשתמש

### שלב 1: תצוגה של המוקאפים
פתח את הקבצים ב-`mockups/` בדפדפן (לחיצה כפולה על הקובץ או `open 01_banners_list.html`).

**המוקאפים חיים** - הקרוסלה בפועל מתחלפת כל 4 שניות, תוכל לראות את ההתנהגות המדויקת.

### שלב 2: קריאת המפרט
פתח את `docs/01_product_spec.md`. זה המסמך הכי חשוב - מכיל:
- פלטת צבעים מדויקת (hex codes)
- טיפוגרפיה וגדלים
- עקרונות עיצוב
- מפרט של כל חלק בממשק
- מפרט מלא של קרוסלת נותני שירות
- פיצ'רים מתקדמים (A/B, Command Palette, Gemini)

### שלב 3: הפעלת Claude Code
1. פתח את `docs/02_claude_code_prompt.md`
2. העתק את כל התוכן
3. פתח טרמינל בתיקיית הפרויקט שלך
4. הרץ `claude` (Claude Code CLI)
5. הדבק את הפרומפט
6. Claude Code יקרא את הקבצים ויתחיל את המימוש בשלבים

### שלב 4: ליווי המימוש
Claude Code יבצע את העבודה בשלבים ויבקש את האישור שלך אחרי כל שלב. זה בכוונה - כדי שתוכל לבדוק ולכוון.

## הפיצ'ר החדש: קרוסלת נותני שירות

זה הפיצ'ר הדגל. איך זה עובד:
1. במסך הבנייה אתה בוחר 2-20 נותני שירות מהרשימה
2. מגדיר משך הצגה (2-8 שניות) - ברירת מחדל 4
3. בוחר סדר: חכם/אקראי/דירוג/ידני
4. בוחר מה להציג בכרטיס (תמונה, דירוג, גלריה, זמינות...)
5. בוחר אנימציית מעבר
6. באפליקציה - הבאנר מתחלף אוטומטית, לחיצה = דף הפרופיל

## עקרונות עיצוב מרכזיים

- **Monochrome + accent יחיד**: שחור מעודן (#0A0A0A) + סגול (#6B5CFF)
- **Two weights only**: 400 ו-500, אף פעם לא 600/700
- **Sentence case תמיד**: אף פעם לא ALL CAPS או Title Case
- **0.5px borders**: גבולות עדינים
- **אין shadows**: רק focus rings
- **micro-interactions**: transitions של 120-180ms

המראה יהיה ברמה של Linear, Stripe, ו-Vercel.

## AI

המערכת משתמשת ב-**Gemini** (לא Claude) - כפי שכבר קיים באפליקציה שלך.
כל התובנות והצעות ה-AI מנותבות דרך ה-Gemini API שלך.
