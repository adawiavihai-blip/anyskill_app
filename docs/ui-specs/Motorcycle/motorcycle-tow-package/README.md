```
motorcycle-tow-package/

    ├── 01-admin-dashboard.html       — פאנל ניהול (אתה כמנהל)

    ├── 03-customer-profile-view.html — תצוגת פרופיל ללקוח
    ├── 04-booking-flow.html          — זרימת הזמנה (5 שלבים)
    └── 05-live-tracking.html         — מסך מעקב חי
```

## איך להשתמש

### שלב 1: בדוק את המוקאפים
פתח כל קובץ HTML בדפדפן כדי לראות את העיצוב המלא. כל המוקאפים מותאמים למובייל ועובדים בעברית RTL.

### שלב 2: שלח לקלוד קוד
1. הורד את כל החבילה
2. פתח את הפרויקט שלך בקלוד קוד
3. שלח את הפקודה הבאה:

```
אני שולח לך חבילה מלאה של מוקאפים ומפרט לפיצ'ר חדש.
קרא את הקובץ PROMPT_FOR_CLAUDE_CODE.md תחילה — שם נמצאות כל ההוראות.
אחר כך עבור על תיקיית mockups/ לפי הסדר המספרי, ובסוף ראה את motorcycleTowDefaults.js לערכי ברירת מחדל.

תתחיל עם הקוד אחרי שתקרא את הכל ותוודא שאתה מבין את המבנה. אם יש לך שאלות לפני שאתה מתחיל - שאל אותי.
```

### שלב 3: תן לקלוד קוד לעבוד
קלוד קוד יקרא את הקבצים, יבנה את הקוד לפי הקיים בפרויקט שלך, ויחבר את הפיצ'ר לאפליקציה הקיימת.

## מה כלול

✅ **4 ממשקים שלמים** עם UI/UX מלא
✅ **פלטת צבעים מוגדרת** (סגול, ירוק, חום, אדום)
✅ **עברית RTL** בכל מקום
✅ **לשון זכר** לנותני שירות
✅ **מפרט טכני מלא** עבור React + Tailwind
✅ **רשימת מה שלא לבנות** (פיצ'רים שכבר קיימים)
✅ **מבנה תיקיות מוצע**
✅ **ערכי ברירת מחדל** לכל הפרמטרים

## מה הפיצ'ר עושה

תת-קטגוריה חדשה "גרר אופנועים" תחת "שירותי רכב ותחבורה". כשנותן שירות בוחר אותה:
1. **מקבל בלוק חכם בעריכת הפרופיל** — ממלא פעם אחת
2. **הבלוק מסתנכרן אוטומטית לפרופיל הציבורי** שהלקוח רואה
3. **לקוחות יכולים להזמין** דרך זרימת 5 שלבים
4. **מעקב חי** מההזמנה ועד הגעה ליעד

הכל בעברית, RTL, מותאם למובייל.
