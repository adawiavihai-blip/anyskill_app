## מה זה הפיצ'ר

הוספה לקטגוריית "גרר אופנועים" הקיימת. הכפתור "מצא גרר דחוף" כבר קיים בעמוד אבל לא עושה כלום — אנחנו מחברים אותו לזרימה חדשה:

**הלקוח** לוחץ → ממלא תקלה + מיקום (מפה אינטראקטיבית כמו וולט) → הקריאה משודרת לכל הגרריסטים ברדיוס → מקבל הצעות → בוחר.

**הגרריסט** מקבל התראה → רואה את הקריאה בלשונית "הזדמנויות" הקיימת → המערכת מחשבת את המחיר אוטומטית לפי הפרופיל שלו → הוא קובע רק ETA → מאשר.

## הייחוד

✅ **הגרריסט לא קובע מחיר** — המערכת מחשבת מהפרופיל שלו
✅ **הגרריסט קובע רק ETA** — תוך כמה דקות יגיע
✅ **לקוח לא רואה פרטי קשר** עד הבחירה
✅ **כרטיסי פרופיל קיימים** — לא עיצוב חדש להצעות
✅ **לשונית "הזדמנויות" קיימת** — רק רכיב חדש בתוכה
✅ **שכבות התראות** — 5km → 10km → 15km בהדרגה (פחות רעש)

## מה קלוד קוד יבנה

- 6 מסכים חדשים (אבחון, מיקום, המתנה, הצעות, כרטיס גרריסט בהזדמנויות, דיאלוג בטיחות)
- 3 שירותים (FlashAuctionService, PricingService, DispatchService)
- 2 Cloud Functions (dispatch + offer-received)
- מודלים + Firestore rules
- חיווט הכפתור הקיים + הוספה ללשונית הזדמנויות
