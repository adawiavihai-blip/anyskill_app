# 🎵 Sound Studio — Mockup Package

חבילת מוקאפים מלאה לטאב "צלילים" המשודרג של האפליקציה.

## מה יש בתיקייה

```
sound_studio_mockups/
├── index.html              ← מסך 1: סטודיו (מיפוי אירועים → צלילים)
├── library.html            ← מסך 2: ספרייה (מאגר כל הצלילים)
├── analytics.html          ← מסך 3: אנליטיקס (KPI, גרפים, תובנות)
├── logs.html               ← מסך 4: לוג מערכת (היסטוריה ובריאות)
├── assets/
│   └── styles.css          ← מערכת עיצוב משותפת
├── CLAUDE_CODE_PROMPT.md   ← פרומפט מלא לקלוד קוד
└── README.md               ← הקובץ הזה
```

## איך לפתוח את המוקאפים

### אפשרות 1 — פתיחה ישירה בדפדפן
לחץ פעמיים על `index.html` ואז נווט בין הטאבים בראש המסך.

### אפשרות 2 — שרת מקומי (מומלץ)
```bash
cd sound_studio_mockups
python3 -m http.server 8000
# פתח בדפדפן: http://localhost:8000
```

## איך להעביר לקלוד קוד

1. ארוז את כל התיקייה לקובץ ZIP
2. פתח שיחה חדשה עם Claude Code בפרויקט הפלאטר שלך
3. העלה את ה-ZIP
4. הדבק את התוכן של `CLAUDE_CODE_PROMPT.md` בהודעה הראשונה
5. תן לקלוד קוד לעשות את העבודה

## מה כל מסך מראה

### מסך 1: סטודיו
- היומיום של אדמין — לראות איזה צליל מתנגן באיזה אירוע
- 5 שורות, אחת לכל `AppEvent`
- כפתור Play בכל שורה — preview מיידי
- Dropdown לבחירת צליל אחר
- אינדיקטור בריאות מערכת בראש

### מסך 2: ספרייה
- מאגר כל הצלילים (לא רק 4 הפעילים)
- העלאת קבצים חדשים
- סינון לפי סטטוס (פעיל/בארכיון/הצעת AI)
- תצוגת עומק עם waveform, סטטיסטיקה, וטביעת רגש

### מסך 3: אנליטיקס
- KPI: סך השמעות, CTR, השתקות, צליל מוביל
- גרף 7 ימים מפולח לפי סוג צליל
- דירוג ביצועים
- תובנת AI עם פעולה מוצעת

### מסך 4: לוג מערכת
- 4 health cards בזמן אמת
- היסטוריית כל הפעולות
- סינון לפי סוג
- כפתור יצוא ל-CSV

## איך זה מתחבר לקוד הקיים

| מסך | Firestore Collection | סטטוס |
|------|---------------------|--------|
| סטודיו | `app_settings/event_sounds` (קיים) | קריאה+כתיבה |
| סטודיו | `app_settings/sounds` (קיים) | קריאה+כתיבה |
| ספרייה | `sound_metadata` | חדש — צריך ליצור |
| אנליטיקס | `sound_events_log` | חדש — צריך ליצור |
| לוג | `sound_system_log` | חדש — צריך ליצור |

הארכיטקטורה הקיימת (`AudioService`, `AppSound` enum, `AppEvent` enum, pre-buffering, iOS unlock) **נשמרת לחלוטין**.
