## 📍 מתי הבלוק החדש מופיע?

הבלוק "הגדרות הדברה" מופיע **רק** במצב הבא:
- המדביר נכנס לעריכת פרופיל
- בחר תת-קטגוריה "הדברה" (`subcategory == 'pest_control'`)

**אם המדביר מבטל את בחירת "הדברה":**
- הבלוק נעלם מה-UI
- אבל הנתונים נשמרים ב-Firestore (במקרה שיחזור)

## 💾 Save Logic

```javascript
async function savePestControlProfile(userId, formData) {
  const pestControlProfile = {
    licenses: formData.licenses,

    pestTypes: formData.selectedPestTypes,

    treatmentMethods: formData.selectedMethods,
    customerTypes: formData.selectedCustomerTypes,

    availability: {
      emergencyService: {
        enabled: formData.emergencyEnabled,
        additionalFee: formData.emergencyFee || 0,
      },
      available247: formData.available247,
      averageArrivalTime: formData.arrivalTime || 45,
    },

    serviceArea: {
      radiusKm: formData.serviceRadius || 30,
      travelFee: formData.travelFee || 0,
      freeRadiusKm: formData.freeRadius || 15,
    },

    basePricing: formData.pricing,

    warrantyAndService: {
      warrantyMonths: formData.warrantyMonths || 3,
      digitalReport: formData.digitalReport ?? true,
      beforeAfterPhotos: formData.beforeAfterPhotos ?? true,
    },

    maintenancePackages: formData.packages.filter(p => p.enabled),

    // 🆕 חדש!
    treatmentInstructions: {
      structuredInstructions: formData.instructions
        .filter(i => i.enabled)
        .map(i => ({
          id: i.id,
          type: i.type,
          icon: i.icon,
          titleHe: i.titleHe,
          enabled: true,
          duration: i.duration,
          color: i.color,
        })),
      customInstructions: formData.customInstructions || '',
    }
  };

  // Validation
  if (!pestControlProfile.licenses.find(l => l.type === 'ministry_environmental_protection' && l.verified)) {
    throw 'נדרש רישיון משרד הגנת הסביבה מאושר';
  }

  if (pestControlProfile.pestTypes.length === 0) {
    throw 'בחר לפחות סוג מזיק אחד';
  }

  if (pestControlProfile.pestTypes.includes('snakes')) {
    if (!pestControlProfile.licenses.find(l => l.type === 'snake_catcher' && l.verified)) {
      throw 'לכידת נחשים דורשת רישיון לוכד נחשים מוסמך';
    }
  }

  if (pestControlProfile.treatmentMethods.length === 0) {
    throw 'בחר לפחות שיטת טיפול אחת';
  }

  // Save
  await firestore.collection('users').doc(userId).update({
    pestControlProfile: pestControlProfile,
  });

  // Trigger sync to public profile
  await syncPestControlProfileToProviderListings(userId);
}
```

## ⚠️ Edge Cases & Validation

1. **המדביר לא בחר/ה תת-קטגוריה "הדברה"** → הבלוק לא מופיע
2. **המדביר ביטל/ה את "הדברה"** → הבלוק נעלם, אבל הנתונים נשמרים
3. **ללא רישיון משרד הגנ"ס** → אי אפשר להגדיר את עצמו כפעיל
4. **לכידת נחשים סומנה ללא רישיון** → אי אפשר לשמור
5. **שום סוג מזיק לא סומן** → אי אפשר לשמור
6. **שום שיטת טיפול לא סומנה** → אי אפשר לשמור
7. **חירום מופעל ללא תוספת מחיר** → ברירת מחדל ₪150
8. **שדה הוראות אישיות מעל 500 תווים** → לא מאפשר להמשיך לכתוב
