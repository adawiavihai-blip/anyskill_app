- לקרוא את כל 5 הקבצים
- לבנות את כל ~30 הקבצים בסדר נכון
- לעקוב אחר חוקי הברזל

- לפרוס את ה-Cloud Functions
- לעדכן `CLAUDE.md` §42

### צעד 3: ⏱️ חכה 8-12 שעות

Claude Code יעבוד באופן עצמאי. הוא ייצור:
- 16 widgets (Provider + Client)
- 5 modals לעריכה
- 6 models
- 3 Cloud Functions
- עדכון CLAUDE.md
- 0 issues ב-flutter analyze

### צעד 4: 🚀 Deploy

```bash

firebase deploy --only functions

# Firestore Rules
firebase deploy --only firestore:rules

# Web app
flutter build web --release
firebase deploy --only hosting
```
