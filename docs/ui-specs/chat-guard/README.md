# Chat Guard AI - מערכת הגנה מתקדמת לצ'אט

מערכת זיהוי וחסימה של ניסיונות עקיפת תשלומים בצ'אט, ברמה של Enterprise 2026.

## 📁 מבנה התיקיות

```
chat-guard/
├── frontend/          # ממשק המשתמש (HTML + JS)
│   ├── admin-dashboard.html    # דשבורד אדמין עם ניהול מילים
│   └── chat-guard.js           # לוגיקת צד-לקוח
│
├── backend/           # לוגיקה של צד שרת
│   ├── detection-engine.js     # מנוע הזיהוי - הלב של המערכת
│   ├── api-routes.js           # נקודות קצה (endpoints)
│   └── risk-scorer.js          # מחשב ציון סיכון
│
├── database/          # מבנה בסיס הנתונים
│   └── schema.sql              # טבלאות של המערכת
│
└── docs/              # תיעוד
    ├── INTEGRATION.md          # איך לשלב במערכת הקיימת
    ├── CLAUDE-PROMPT.md        # הנחיות ל-Claude שיבנה את זה
    └── API.md                  # תיעוד ה-API
```

## 🚀 התחלה מהירה

### שלב 1: העבר את כל התיקייה לפרויקט שלך

### שלב 2: תן ל-Claude את הקובץ `docs/CLAUDE-PROMPT.md`
זה מכיל את ההנחיות המלאות - Claude יבנה את המערכת במערכת הקיימת שלך.

### שלב 3: הגדר את בסיס הנתונים
הרץ את `database/schema.sql` במערכת הנתונים שלך (Firebase/Supabase/MySQL - תתאים בהתאם)

### שלב 4: פתח את `frontend/admin-dashboard.html` בדפדפן
זה הדשבורד המלא עם כל הפיצ'רים.

## ✨ פיצ'רים עיקריים

- ✅ הוספת/עריכת/מחיקת מילות מפתח בזמן אמת
- ✅ זיהוי הסוואות (רווחים, אימוג'ים, Leetspeak)
- ✅ זיהוי סמנטי (הבנת כוונה)
- ✅ 5 רמות חומרה (אזהרה רכה → השעיית חשבון)
- ✅ Risk Score לכל משתמש
- ✅ A/B Testing של כללים
- ✅ Graph Analysis לזיהוי רשתות הונאה
- ✅ Dashboard מלא עם אנליטיקס
- ✅ API מלא לשילוב במערכת

## 🔗 סנכרון

כל הנתונים נשמרים בבסיס הנתונים ומסונכרנים בזמן אמת. כל שינוי בדשבורד:

- משפיע על הזיהוי בכל צ'אט באפליקציה
